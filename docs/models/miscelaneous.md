## columnsNames

What if you want to get all the column names? Maybe to check if the a certain relation has a certain property?

```javascript
  Coffee.columnsNames();
```

will return an array with the name of all the columns: 

```javascript
  [ 'id', 'name', 'price', 'created_at', 'updated_at' ]
```
